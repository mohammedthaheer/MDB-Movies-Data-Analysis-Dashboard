# IMDB Movies Data Analysis (SQL + Power BI)

## Project Overview
This project analyzes an IMDB movie dataset to identify trends in movie popularity, box office performance, and director contributions. The analysis combines SQL for data querying and Power BI for interactive data visualization.

The goal of this project is to demonstrate how data analytics tools can uncover insights in the entertainment industry.

---

## Dataset
The dataset contains information about 47 movies including:

- Movie titles
- Release dates
- Ratings
- Vote counts
- Directors
- Revenue information

---

## Tools Used
- SQL (Data querying and filtering)
- Power BI (Dashboard creation and visualization)
- Data Visualization techniques

---

## Dashboard Insights

### Total Movies
The dataset contains **47 movies** analyzed in this project.

### Most Popular Movies
Based on vote counts:
1. Jurassic World
2. Captain America: Civil War
3. Avatar

### Most Bankable Movies
Based on revenue performance:
1. Avatar
2. Titanic
3. The Avengers

### Most Awarded Movie
The movie with the highest rating in the dataset is:
- The Dark Knight Rises

### Directors Who Made the Most Movies
The top directors contributing the most movies include:
- Gore Verbinski
- Peter Jackson
- Sam Raimi

### Most Bankable Director
James Cameron generated the highest total revenue from his films in the dataset.

---

## Key Insights
- Box office revenue and audience popularity do not always align.
- Certain directors consistently produce high-performing movies.
- Data visualization helps reveal patterns in movie success and audience engagement.
